process is already running or crashed

启动 redis 服务时，可能遇到完整提示如下：

这个问题常见于服务器异常奔溃，主要是 redis 服务未正常关闭导致的

解决办法也很简单，/work/servers/redis/status/redis6379.pid 文件内容就是奔溃前 redis 的进程号，将它删除重启即可

当然这是在数据不重要的情况下，如果需要执行数据恢复就另说了。
